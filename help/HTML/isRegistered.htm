isRegistered - web automation property

Not available in Open Twebst edition!
Determines whether Twebst web automation software is currently registered.

Applies to:
Core
  var r = coreObj.isRegistered;
Arguments:
None.
Remarks:
The property is true if Twebst web automation software is registered, false otherwise. Registration is done on a per-user basis. A registration key is needed for every Windows user. When the product is registered, all PRO features are enabled and no demo warning dialog boxes are displayed.
Example:

None.
