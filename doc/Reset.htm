Reset - web automation method

Resets web automation core object.

Applies to:
Core
  coreObj.Reset();
Arguments:
None.
Remarks:
Resets all properties of web automation core oject to their default values.
Example:
var coreObj = new ActiveXObject("OpenTwebst.Core");
coreObj.Reset();
