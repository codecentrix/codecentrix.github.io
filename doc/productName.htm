productName - web automation property

Returns the name of the installed Twebst web automation software.

Applies to:
Core
  var name = core.productName;
Arguments:
None.
Remarks:
Returns: "Open Twebst"
Example:
var coreObj = new ActiveXObject("OpenTwebst.Core");
var name    = coreObj.productName;
WScript.Echo(name);
